# Premium Registration Form

A modern, highly functional, and responsive registration form featuring a sleek glassmorphism design, real-time validation, and advanced user experience features.

## ✨ Features

-   **Premium Glassmorphism UI**: A stunning design with a dynamic geometric grid background and grainy textures.
-   **Dark/Light Mode Support**: Seamlessly toggle between dark and light themes with preference persistence.
-   **Intelligent Validation**:
    -   **Name Validation**: Strictly checks for valid names (letters and spaces only).
    -   **Auto-Capitalization**: Automatically transforms name inputs into proper Title Case (e.g., `john doe` → `John Doe`).
    -   **Anti-Gibberish Bio**: Ensures the "Bio" field contains meaningful text (min 10 characters, at least one space, and no repeated sequences).
    -   **Real-time Feedback**: Visual error states with shake animations and bold error messages.
-   **Password Security**:
    -   Dynamic **Password Strength Meter** with color-coded levels.
    -   Password **Visibility Toggle** (Show/Hide).
-   **Fully Responsive**: Optimized for all screen sizes, from mobile phones to ultra-wide monitors.
-   **Simulated Submission**: Includes a loading state and a beautiful success animation upon registration.

## 🛠️ Technology Stack

-   **HTML5**: Semantic structure for better accessibility.
-   **Vanilla CSS**: Custom properties (variables), Flexbox, Grid, and keyframe animations.
-   **Vanilla JavaScript**: Clean, modular code for all interactive features and validation logic.
-   **Google Fonts**: Uses the "Inter" typeface for a modern look.

## 🚀 Getting Started

1.  Clone the repository or download the files.
2.  Open `index.html` in any modern web browser.
3.  Fill out the form and experience the premium interactions!
